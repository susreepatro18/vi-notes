# Vi-Notes: Authenticity Verification Platform

Vi-Notes is a specialized writing environment that ensures content is human-generated by monitoring typing dynamics (rhythm, speed, and behavior). The project has been fully implemented and pushed to [GitHub](https://github.com/susreepatro18/vi-notes.git).

## Features Overview

### 1. Premium Dark-Mode Editor
A distraction-free writing space with high-end typography and glassmorphism aesthetics.
![Vi-Notes UI Mockup](/C:/Users/susre/.gemini/antigravity/brain/f53cef4d-c035-48fe-bd31-5bc3ac74061b/vinotes_ui_mockup_1774348534401.png)

### 2. Live Authenticity Tracking (Writing Lab)
The editor features a real-time **Behavioral Log** and a **Human Confidence** meter.
- **Rhythm Analysis**: Continuous monitoring of keystroke intervals.
- **Dilution & Recovery**: A sophisticated model where manual typing can "earn back" confidence points after a paste event.
- **Real-time Logs**: Visual feedback on every significant behavioral event (pauses, pastes, deletions).

### 3. Advanced Analysis Engine
The system implements a hardware-level entropy calculation:
- **Dwell Time & Variance**: Detects the "jitter" in human typing that bots cannot easily replicate.
- **Revision Patterns**: Recognizes iterative thinking through deletion and modification events.

### 4. Authorship Certificates
Downloadable PDF reports that serve as behavioral proof of work.
- **Confidence Score**: Clear pass/fail metric.
- **Forensic Snapshots**: Includes a sample of the behavioral logs.
- **Digital Fingerprint**: Unique identifier for session verification.

### 5. Full-Stack Integration
- **Frontend**: React + TypeScript + Vite (Port 5173).
- **Backend**: Node.js + Express (Port 5000).
- **Database**: MongoDB (Anonymized Keystroke pairs).
